Product Custom Options

No more dependency on variants for internal custom categories in Odoo!

Note : Why to manage separate inventories for variants when you can create custom options for products in Odoo; instead of Product variants!

Product Custom Options taking over Variants

UnLike variants, you don’t need to manage the inventory of category custom options. Hence, simply select the options while creating sales orders.

Create Custom Options for Product in Odoo

You can create common custom options for the product categories. No need to manage multiple stock for custom options. Custom options are applicable to all the products which come under that category.

Features

Below is the detailed list of Feature for Odoo Custom Option Module

Detailed navigation panel/menu with product preview animations

  • No need to manage the stock for variants of a product separately
  • Product variations do not dependent on attributes

Simple Steps to define Custom Products

  • A separate section to manage custom options
  • Define custom options directly from the product view

Set Input Methods to add custom options in order line

  • multiple input selection types are supported
  • Set dropdown menu, radio button, enter time, date text or upload a file

Select Product Options in the Order line

  • Shows all the selected options of a product on a single order line in the sales order
  • You can upload a file if respective custom option is added
  • The price automatically updates as per the chosen category.

Product Custom options tab

User can easily add multiple custom options in product category.

Custom Options tab in Product Section

Enable the 'Use Category Options’ checkbox and add the appropriate category in product template.

Custom options configuration

User can select and add custom options as per the choice. You can also set Price for each option.

Custom options in Sale Order

You can see a Configuration button which will direct you to the configured custom options.

Other Modules

Odoo Category Custom Options
  • Using this module you can add custom options to the category which will be adopted by new product created under this category.
  • Product variations that do not rely on attributes.
  • Admin can easily define custom options from product view.
  • Many types of options are supported like Text, Date, Radio, Dropdown, Checkboxes & file, etc. Multiple input selection types are also supported.
  • You can also provide a custom option to upload a file.

Odoo Website Custom Options
  • Reduced individual product inventory management efforts, flexible product flow, and higher efficiency- that's what "Odoo Website Custom Options" module aims for.
  • Product variations that do not rely on attributes.
  • Single inventory of the product, unlike Variants.
  • Admin can easily define custom options from product view.
  • Many types of options are supported like, Text, Date, Radio, Dropdown & file etc.
  • Multiple input selection types are also supported like Checkbox & Multiselect.
  • You can also provide a custom option to upload a file.
