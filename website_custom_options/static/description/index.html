Odoo Website Custom Options

Let Your Customers Customize the Products!

Make The Products Fully Customizable For Your Customers

You can create customizable products using the Custom Options. The best part is that custom options work just like variants, however, you don’t need to manage stocks unlike variants.

Offer Multiple Input Options to Your Customers

Set appropriate input methods such as the dropdown menu, radio button, enter time, date text or upload a file to add a custom option in an order line. So that the customers can customize the products in the way they want.

Let Customers’ pick the Custom Options Directly From Odoo Website

The module allows you to offer the customizable products in Odoo Website. Your customers need to simply pick the options and customize the product in the way they want.

Understand Odoo Website Custom Options With an Example

Suppose, you want to run a bakery online; then creating variants for all possible customization is not feasible. In this case, you can create custom options in place of attributes and let your customers decide what needs to be prepared.
Set unique inputs for different products like dropdown, multi-select, checkboxes and even file upload for perfectly customized customer experience.
Suppose, a customer wants to add a picture on his/her cake then he/she can use the option to upload files; similarly, text to add messages and so on.

What do you get with Odoo Website Custom Options?

  • Show Custom Options in Odoo Website.
  • No need to manage the stock for variants of a product separately.
  • It provides a simple interface to add custom options of the product on the website product page.
  • Custom options will be shown just like the variant attribute selection.
  • Set appropriate input methods such as the dropdown menu, radio button, enter time, date text or upload a file to add a custom option in an order line.

Other Modules

Odoo Category Custom Options
  • Using this module you can add custom options to the category which will be adopted by new product created under this category.
  • Product variations that do not rely on attributes.
  • Admin can easily define custom options from product view.
  • Many types of options are supported like Text, Date, Radio, Dropdown, Checkboxes & file, etc. Multiple input selection types are also supported.
  • You can also provide a custom option to upload a file.

Odoo Product Custom Options
  • Product Custom Options taking over Variants
  • UnLike variants, you don’t need to manage the inventory of category custom options. Hence, simply select the options while creating sales orders.
  • Create Custom Options for Product in Odoo.
  • You can create common custom options for the product categories. No need to manage multiple stock for custom options. Custom options are applicable to all the products which come under that category.
